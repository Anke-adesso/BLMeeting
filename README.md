# BLMeeting — Claude for adesso MS

A print-ready handout for the adesso BL meeting introducing Claude AI to colleagues.

## File

**`claude_handout_overall-2.html`** — Single-page A4 handout covering:
- What Claude is and how it works
- Key use cases for consulting work
- Power features (Projects, long context, MCP add-ins)
- Claude vs. ChatGPT comparison
- Practical tips and access info

## Usage

Open the HTML file in a browser. Use the **Print** button in the top bar to export as PDF (A4, no margins).

Toggle between **English and German** using the language button in the top bar.

## Access at adesso

Request access via **matrix24**, then log in at [claude.ai](https://claude.ai) via adesso SSO.
Add-ins are available for Word, Outlook, Excel, and PowerPoint.
